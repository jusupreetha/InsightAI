# InsightAI

AI-Powered Data Analytics Platform

InsightAI is an AI-powered web application that transforms CSV and Excel datasets into meaningful insights through interactive visualizations and Google Gemini AI. It enables users to explore data, generate analytical reports, and gain actionable insights without writing code.

**Live Demo:** https://insightai-analytics.streamlit.app

**Tech Stack:** Python, Streamlit, Pandas, Plotly, Google Gemini AI, ReportLab
